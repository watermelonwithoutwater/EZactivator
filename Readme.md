# EZactivator PRO 🚀
Advanced Windows Activation Tool with Stripe API Integration.

## Features ✨
- **Secure Login:** Integrated with SQLite database.
- **Stripe Payments:** Automated license activation via Stripe API.
- **Smart Automation:** Uses `pyautogui` for seamless system activation.
- **UAC Ready:** Built-in administrative privilege requests.

## How to use? 🛠️
1. Download the `EZactivator_Setup.exe` from the [Releases](link-to-your-release) section.
2. Install the application.
3. Login or Create an account.
4. Enter Promo Code `EZ2026` or complete the payment to unlock the Pro menu.
5. Click **ACTIVE NOW** and let the magic happen!
